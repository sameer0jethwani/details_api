

download this project from github-
than go to the folder details_api-master(project folder)
than open terminal in that same folder
please install these from terminal , (if required)-
pip install fastapi ,uvicorn ,sqlalchemy
than run command - uvicorn details_app:app --reload
once you do this ,it will create a database or connect data base- address_database.db
than go to your browser and open - http://127.0.0.1:8000/docs
now you can play with the API and all details will be saved in database :-)
